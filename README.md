## Описание

Bandwagon — веб-приложение для поиска похожих музыкальных исполнителей. Введите имя любимого артиста и откройте для себя новую музыку через рекомендации Last.fm API.

## Технологии

- React + TypeScript
- Vite
- Tailwind CSS
- Last.fm API
